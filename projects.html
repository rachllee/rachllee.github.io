Class Projects

Minesweeper in Java

For my CIS 120 final project, in which we were instructed to create any game we wanted, I recreated Minesweeperd. Functionalities included the ability to uncover squares, flag and unflag, save progress, see remaining flags with a live progress bar, "always safe first click", and restart. I also drew out the flags and squares within Java Swing.

Paint in OCaml

For one of my CIS 120 project assignments, I was given a primitive OCaml GUI toolkit and paint program. I used this to create a paint application, implementing features such as shape tools (for lines and ellipses), an undo button, the ability to thicken lines, and a color slider.

Personal Projects

This Website!

I used the summer of 2023 as an opportunity to create a personal website showcasing my skills and projects, as well as to gain some experience in HTML, CSS, and Javascript. This website is fully web responsive, and adapts to different screen sizes and devices. It also contains an unfolding navigation bar, interactive buttons that respond to a user's hover and click, a color changing title, and a dynamic dark mode (a preference that is stored locally).

Created by Rachel Lee with HTML5/CSS and Javascript
